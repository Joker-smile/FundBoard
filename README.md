# 📈 场外指数基金多源数据监控工具

一款为投资者量身定制的智能化、现代化、多数据源的场外指数基金监控与盯盘桌面应用程序。不仅能实时追踪主要海外指数及国内宽基场外基金，更支持大A指数实时行情、右键便捷自选管理，以及交易状态后台定时轮询与邮件报警提醒，助您完美把握申购与定投时机。

## 🎯 详细功能列表

本软件提供了极其丰富的交互、数据分析与自动化预警功能：

### 1. 核心数据与多源获取
- **多数据源无缝切换**：聚合了**天天基金**（默认）、**蚂蚁基金**以及**晨星数据**的接口，确保数据高可用。
- **自定义指数追踪**：在设置中支持通过关键词（如“纳斯达克”、“红利”）自定义追踪的场外指数基金。
- **后台异步静默刷新**：支持手动和**定时自动刷新**（如每5/10/30/60分钟），获取数据时进度条显示，主界面永不卡顿。
- **大A指数实时盯盘**：状态栏右侧集成了**沪深创业科创50**四大主要指数，后台独立线程 **3秒一次极速轮询刷新**，并根据涨跌呈现红/绿视觉效果。

### 2. 强大的交互与浏览操作
- **双击直达内嵌网页**：采用 Edge Chromium 内核渲染，双击某只基金即可直接弹出独立的高清网页窗口，浏览基金深度详情，无需跳出应用。
- **历史净值可视化图表**：在基金上右键选择“查看历史净值图表”，即可调出过去30天的净值走势折线图，支持鼠标悬停查看数值和图表保存。
- **支持多选与右键菜单**：按住 `Ctrl` 或 `Shift` 点击，或者直接拖拽可同时选中多只基金；右键菜单提供专属操作入口。
- **右键智能自选管理**：支持右键直接选择 `⭐ 加入自选` 或 `🗑️ 取消自选`。程序拥有智能安全删除机制：非其他分类标签的基金取消自选时将从库中彻底删除；属于其他指数分类的基金取消自选时仅清除自选标识，防止主列表数据被误删。
- **“⚡ 更新选中” (局部极速刷新)**：选中特定的基金后，点击工具栏或右键菜单的“更新选中”，2秒内精准刷新所选基金的交易状态和最新净值。

### 3. 数据过滤与检索引擎
- **多词模糊搜索**：搜索框支持用**空格分词**（例如输入 `广发 纳指 A`），即可精准、模糊地找出名称中同时包含这些词的基金或代码。
- **交易状态多维过滤**：一键筛选“暂停赎回”、“限大额”等多种具体的交易限制状态，是决定今日能否加减仓的得力助手。
- **表头动态排序**：点击任意表头（如日增长率、单位净值），即可对当前列表进行升序/降序排列。

### 4. 交易状态邮件监控与预警
- **后台自动静默轮询**：程序会在后台定时检查所有自选基金最新的申购限额与交易状态（如开放、暂停大额、限额额度调整等）。
- **UI 界面化邮箱配置**：在「设置」窗口新增 **「邮件监控」选项卡**，支持直接在 UI 界面配置启用状态、SMTP 发信设置（服务器、端口、发信账号、授权码）、收件人邮箱，以及**定时轮询的监控间隔时间（分钟）**。
- **状态变更即时通知**：一旦发现自选基金中出现了新的交易状态，程序会自动更新本地数据库，并由后台发信线程立刻向您的邮箱发送一封详细的状态变更通知邮件，避免错过限购额度调整或买入机会。

### 5. 极致现代化护眼 UI (防疲劳)
- **Superhero / Flatly 动态主题**：夜间模式采用专业金融终端都在用的深蓝灰（Superhero）色调，白天模式采用清新的扁平轻量（Flatly）色调，降低长久盯盘的视觉疲劳。
- **红涨绿跌与呼吸感排版**：符合中国股市习惯的涨跌着色，30px 拉宽行高，搭配暗色斑马纹自适应排版，坚决防止看错行。

### 6. 数据留存与分析
- **本地 SQLite 强缓存**：每次获取数据自动备份，下次启动程序秒开并加载最后一次的缓存数据。
- **一键导出为 Excel/CSV**：随时把当前筛选好、排序好的表格数据导出到本地，供更高级的表格分析工具使用。

## 📊 展示的基金核心数据维度

1. **基金代码** (`code`)：唯一标识符。
2. **基金名称** (`name`)：完整的基金名称（支持名称与交易状态的自适应拉伸排版）。
3. **交易状态** (`purchase_limit`)：当前该基金的交易限制状态，如“开放申购”、“暂停申购”、“限大额(单日上限100元)”等。
4. **跟踪指数** (`index_type`)：根据您配置的关键词自动归类的所属指数标签。
5. **单位净值** (`nav`)：最新公布的单位净值。
6. **净值日期** (`nav_date`)：净值数据的最新归属日期。
7. **累计净值** (`acc_nav`)：反映基金成立以来的历史总收益表现。
8. **日增长值** (`daily_change`)：相较于上一交易日净值的绝对增减数值（红涨绿跌）。
9. **日增长率(%)** (`daily_change_pct`)：相较于上一交易日的涨跌幅度百分比。

## 🚀 快速开始

### 1. 环境准备

确保您的电脑上已经安装了 **Python 3.8+**，并且安装了以下核心依赖库：

```bash
pip install ttkbootstrap requests matplotlib pywebview openpyxl
```

### 2. 运行程序

在项目根目录下，直接运行主入口文件即可：

```bash
python main.py
```

### 3. 操作指南

- **获取全市场数据**：首次打开程序，点击左上角的 `🔄 获取数据`。
- **配置自选与邮件监控**：
  1. 点击右上角 ⚙️ **编辑 -> 设置**。
  2. 在 **「基础设置」** 中自定义您关注的指数提取关键词（如：`纳指: 纳斯达克, 纳指`）。
  3. 在 **「邮件监控」** 中，勾选启用监控，填写您的邮箱 SMTP 参数和接收邮箱，并设置检查间隔时间（如 `30` 分钟）。
  4. 保存设置，后台交易状态邮件监控系统即时启动。
- **右键快速自选与局部刷新**：
  - 对着列表中任意一只基金右击鼠标，在菜单中可以直接选择 `⭐ 加入自选` 或 `🗑️ 取消自选`。
  - 选中多只重点关注的基金，右击选择更新，或者点击工具栏的 `⚡ 更新选中`，2 秒内单独刷新这几只基金的最新状态。
- **查看历史净值折线图**：右击基金，选择 `📈 查看历史净值图表`。
- **浏览网页级深度详情**：双击任意一行基金，直接弹出该基金在天天基金网上的全貌网页。

## ⚙️ 架构与可执行文件打包

- `gui/`：基于 Tkinter + TtkBootstrap 构建的现代化响应式视图组件与对话框弹窗。
- `data_sources/`：抽象工厂模式设计的数据源拉取逻辑。
- `database.py`：基于 sqlite3 的轻量级本地数据引擎，可打包环境动态路径自适应。
- `config.py`：全局状态、关键字配置、邮件服务与用户首选项管理。

### 📦 Windows 打包可执行文件 (.exe)
1. 安装 PyInstaller：`pip install pyinstaller`
2. 执行打包命令：
   ```bash
   pyinstaller --noconsole --onefile --clean --name="场外指数基金数据获取工具" main.py
   ```
3. 打包完成后，将生成的 `dist/场外指数基金数据获取工具.exe` 与项目中的 `user_config.json` 放在**同一个目录下**，即可双击完美运行。
